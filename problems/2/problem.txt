Balls and Boxes
Intermediate Math
7
1
There are N distinguishable boxes and M distinguishable balls. Find the number of ways to put all M ($1 <= M <= 10^3$) balls into these N ($1 <= N <= 10^3$) boxes and each box should have at least one ball.

## Input Specification ##

The first line of input contains two space-separated integers, N and M.

## Output Specification ##

Output the number of ways satisfy the conditional described above. Modulo your answer by 10^9 + 7.

## Sample Input ##

```
2 4
```

## Sample Output ##

```
14
```
